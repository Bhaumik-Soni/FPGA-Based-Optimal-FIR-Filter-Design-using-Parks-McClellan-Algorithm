# FPGA-Based-Optimal-FIR-Filter-Design-using-Parks-McClellan-Algorithm
Implemented on a Spartan-6 FPGA , this 21-tap low-pass FIR filter uses the Parks-McClellan algorithm for optimal signal smoothing. Features include a Verilog MAC architecture , 16-bit fixed-point coefficients , and an internal frequency divider for real-time oscilloscope verification of harmonic reduction.
